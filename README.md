단순히 자신의 MBTI 유형을 알아보는 것을 넘어 같은 유형의 사람들과 공감하고 다른 유형의 특성을 이해하려는 움직임으로 이어진 것.

간단한 성격 테스트와 이를 이용한 추천 시스템이 인기를 끌고 있다.
매번 질문에 대한 답을 선택하는 귀찮음이 있고 선택하는 과정에서 의식적으로 실제 성격과는 다른 답을 고르는 경우도 있다.
자주 사용하는 SNS를 기반으로 게시글을 분석하여 성격 테스트 결과를 알아볼 수 있다면 좋을 것이다.

## 개발 내용

1. 인스타그램의 특정 계정에서 사진과 텍스트를 크롤링한다.
2. 사진의 색감을 분석하여 그룹화 한다.
	- 게시한 사진의 주요 색상, 채도, 명도를 기준으로 나눈다.
	- 주요 색감의 빈도를 데이터로 얻는다.
3. 텍스트에서 성격을 특정지을 수 있는 단어를 찾는다.
	- 주요 단어의 빈도를 데이터로 얻는다.
4. 얻은 데이터를 기반으로 mbti를 예측한다.
5. 인스타그램 공개 계정 아이디를 입력하면 mbti를 예측하여 결과를 알려준다.

----

가능하다면 두 계정의 mbti를 이용하여 좋은 관계를 맺을 수 있는지 추천하는 기능을 추가한다.
